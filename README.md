# 💰 FinTrack Lite

**FinTrack Lite** — персональный финансовый трекер с автоматическим импортом выписок из Сбербанка (PDF/CSV) и наглядной аналитикой.


### 📊 Дашборд
- KPI-карточки: баланс, доходы, расходы, среднее за месяц
- График доходов/расходов по месяцам (Chart.js с отображением сумм)
- Круговая диаграмма расходов по категориям
- Прогноз расходов на следующий месяц с анализом трендов
- Переводы на цели и счета **не считаются расходами**

### 📋 История операций
- Фильтрация по дате, категории, типу операции
- Группировка транзакций по описанию с раскрывающимися деталями
- Пагинация (50 записей на страницу)
- Защита от дубликатов при повторной загрузке

###  Финансовые цели
- Отслеживание прогресса накоплений (вклады, инвестиции, подушка безопасности)
- Автоматическое обновление сумм из транзакций
- История изменений каждой цели
- Визуализация прогресса с процентами

### 💳 Счета и активы
- Учёт всех финансовых счетов (банковские, брокерские, вклады, наличные)
- Группировка по типам счетов
- Общая сумма активов

### 🔗 Привязка транзакций
- Удобный интерфейс для привязки транзакций к целям и счетам
- Группировка по описанию (одинаковые переводы вместе)
- Массовая привязка одной операцией
- Автоматическое исключение привязанных транзакций из расходов

### 📥 Импорт данных
- Парсинг PDF-выписок Сбербанка (распознаёт доходы/расходы, имена, магазины)
- Импорт CSV из Тинькофф, Сбербанка и других банков
- Авто-определение финансовых целей по ключевым словам

### 🛡️ Безопасность
- Авторизация через Django Admin
- Изоляция данных пользователей
- Валидация входных данных

### ⚙️ CI/CD
- GitHub Actions для автоматического тестирования
- Проверка проекта при каждом пуше (Windows environment)

## 🚀 Быстрый старт

### Требования
- Python 3.10+
- Windows 10/11 (тестировалось на Windows)

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/O777GR/FinTrack-Lite.git
cd FinTrack-Lite

# Создайте виртуальное окружение
python -m venv .venv
.venv\Scripts\activate

# Установите зависимости
pip install -r requirements.txt

# Примените миграции
python manage.py migrate

# Создайте суперпользователя
python manage.py createsuperuser

# Запустите сервер
python manage.py runserver 8080
```

### Очистка базы данных
🗑️ Удалить только транзакции

python manage.py shell

Внутри оболочки Python:

from finance.models import Transaction, FinancialGoalHistory

FinancialGoalHistory.objects.all().delete()
print("✅ История целей удалена")

Transaction.objects.all().delete()
print("✅ Транзакции удалены")

exit()

## 📊 Использование
Загрузите выписку из банка (PDF Сбербанка или CSV)
Создайте цели через админку или на странице "Цели"
Привяжите транзакции к целям/счетам на странице "Привязать"
Следите за прогрессом на дашборде и странице целей
